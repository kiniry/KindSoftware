Hosted Mailing Lists

KindSoftware hosts several mailing lists:

announce
The announce list is used to keep interested parties up-to-date about major changes that take place here at KindSoftware. E.g., a new piece of software is released, a new version of an existing piece of software is released, a major new section of the website is introduced, a new paper is published, etc. This is a very low-volume list with, we suspect, one message per fortnight.
applied-formal-methods
The applied-formal-methods list centers on the primary focus of KindSoftware - that of promoting the use, development, adoption, and advancement of formal methods. Discussions include anything having to do with the application of formal methods in any venue; software, hardware, UI/HCI, pure and applied mathematics, engineering, etc.
software-for-correctness
The software-for-correctness list is used to discuss pieces of software that implement formal methods in some manner. Software of all kinds, developed by KindSoftware or not, academic, commerical, Open Source or proprietary - they are all fair game. Typical posts would include reviews, comparisons, feature discussions, community needs, etc.

You can subscribe and browse to these lists from our Mailing Lists page.
