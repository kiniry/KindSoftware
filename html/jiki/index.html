The KindSoftware Jiki

The Jiki is an open, read-write web architecture built with distributed components and implemented in Java. We'll be putting a new and improved Jiki up in the space in the coming months to support the KindSoftware community through knowledge capture via discussions, archiving, and more. See the Jiki homepage for more information on Jiki.
