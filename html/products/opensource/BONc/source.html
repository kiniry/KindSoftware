BONc Subversion Access

The source code for BONc is available through anonymous subversion access over HTTP at the following url:
https://oss.itu.dk/repos/mobius/src/bon/bonc/bonc

One checks out the source with the following command:
svn co https://oss.itu.dk/repos/mobius/src/bon/bonc/bonc

One can also browse the source in your browser via the Mobius Trac at the following location: https://oss.itu.dk/trac/mobius/browser/src/bon/bonc/bonc.

Please see the installation instructions for further details on how to build the source.
