CLOPS

CLOPS is a Java framework for painless handling of command line options in Java applications. It consists of a code generator and a runtime library.

It is developed by a large team from our research group. The idea sprung from the research in Software Product Lines of Mikoláš Janota. Most of the code is written by Fintan Fairmichael, with substantial contributions from Mikoláš, Viliam Holub and Julien Charles. As usual, everything is done under the careful eye of our guru.

The web home of CLOPS is on SourceForge.
