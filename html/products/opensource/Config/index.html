SATConfig Summary

SATConfig is a tool for performing configurations of given in a Conjunctive Normal Form using a SAT solver. The main ideas behind the tool are described in the article Do SAT Solvers Make Good Configurators?.

SATConfig is being developed by Mikoláš Janota as a part of his PhD research. It is written in Java and will run correctly on any system with a 1.5/5.0 or later virtual machine.

The latest version of SATConfig can be downloaded from the following page:

http://www.kindsoftware.com/products/opensource/Config/releases

Development of SATConfig is hosted at GitHub.
