Javafe

This is the Java frontend which parses Java 1.5 bytecode and Java 1.4 source code. It is used as the compiler foundation of several tools including ESC/Java2 and RCC.

An Eclipse plug-in for Javafe is available via https://www.kindsoftware.com/products/opensource/Mobius/updates/

Development of JavaFE is hosted at GitHub.

The binary release is available in three archive formats: zip tgz tbz

The source release is available in three archive formats: zip tgz tbz
