README for SenseTile

Welcome to SenseTile, a hardware and software framework for large-scale experiments with complex multimedia sensing and processing data at terabyte scales.

Contribution

If you wish to contribute to SenseTile, please email Dr. Joseph Kiniry to discuss collaboration opportunities.

Documentation

This release of SenseTile comes with several pieces of documentation:

  1. The SenseTile brief, a short summary of the framework.
  2. A SenseTile simulator BON-JML examples. See BONc project for more information on BON.
  3. Full Java-Jmldoc-generated documentation of SenseTile Sensor Board driver and simulator.
